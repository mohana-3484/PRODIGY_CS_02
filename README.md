Pixel Manipulation for Image Encryption

Description

This project is a simple Image Encryption and Decryption Tool developed using Python and the Pillow library. The tool encrypts images by manipulating pixel values with a mathematical operation and decrypts them back to their original form using the same key.

The project demonstrates basic encryption concepts, image processing techniques, and pixel-level manipulation, making it a useful beginner-friendly cybersecurity project.

Features

* Encrypt images using pixel value manipulation
* Decrypt encrypted images back to the original image
* Simple and easy-to-understand implementation
* Uses Python and Pillow (PIL) for image processing
* Supports common image formats such as JPG and PNG

Technologies Used

* Python 3
* Pillow (PIL)


How to Run
1. Clone the Repository
2. Install Dependencies
3. Add an Image
4. Run the Program
5. Choose an Option
 Output Files

 Encrypted image
 Decrypted image

Learning Outcomes

* Basics of image encryption and decryption
* Python image processing with Pillow
* Practical application of cybersecurity concepts
